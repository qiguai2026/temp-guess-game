### 🎮 游戏玩法

1. **选择模式**：普通/困难/无尽
2. **输入猜测**：输入一个中文词汇
3. **查看温度**：0-100度表示相似程度
4. **继续猜测**：根据温度提示调整策略
5. **猜中获胜**：温度达到100度即胜利

### 💡 提示系统
- 同类别词汇温度较高
- 字符相似也有温度加成
- 部首相似提供额外线索

### 🚀 运行方式

#### 方式1：微信开发者工具（推荐）
1. 下载安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开开发者工具，点击"小游戏" → "导入项目"
3. 选择 `temp-guess-wx` 目录
4. AppID 选择"测试号"或填写自己的 AppID
5. 点击编译运行

#### 方式2：命令行（需要安装工具后）
```bash
# 进入项目目录
cd temp-guess-wx

# 打开微信开发者工具（安装后）
open -a "微信开发者工具" .
```

### 📱 真机调试
1. 在微信开发者工具中点击"预览"
2. 使用微信扫码在手机上体验
3. 或点击"真机调试"进行调试
