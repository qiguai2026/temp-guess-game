# 温差猜词游戏 - 微信小游戏版 (TempGuess WX)

## 项目概述
将原有的 Node.js 温差猜词游戏移植到微信小游戏平台。

## 已完成内容

### 1. 项目结构
```
temp-guess-wx/
├── game.js              # 游戏入口
├── game.json            # 小游戏配置
├── project.config.json  # 微信开发者工具配置
├── js/
│   ├── main.js          # 主程序/UI渲染
│   ├── game.js          # 游戏逻辑
│   ├── words.js         # 词库（400+词汇）
│   └── similarity.js    # 相似度算法
└── images/              # 图片资源目录
```

### 2. 核心功能移植
- ✅ 词库系统（9大类别，400+词汇）
- ✅ 相似度计算算法（编辑距离、字符重叠、部首相似、类别匹配）
- ✅ 游戏状态管理（普通/困难/无尽模式）
- ✅ 温度计算与显示
- ✅ 猜测历史记录

### 3. UI界面
- ✅ 主菜单（模式选择）
- ✅ 游戏界面（温度显示、输入框、历史记录）
- ✅ 温度条可视化（渐变色）
- ✅ 游戏结束画面（胜利/失败）
- ✅ 帮助说明页面

### 4. 交互功能
- ✅ 触摸事件处理
- ✅ 输入对话框
- ✅ 提示系统
- ✅ 历史记录查看
- ✅ 放弃游戏

## 技术栈
- 微信小游戏框架
- ES6 模块化
- Canvas 2D 渲染
